## P1 — 字段全按后端对齐

### 类型定义

- [ ] **`src/types/user.ts`**
  - `User` 接口：`username` → 移除，用 `openid`；`register_time` → `created_at`；`status` → `is_active: boolean`
  - `UserDetail` 接口：同步上面改动，`register_time` → `created_at`
  - `UserFilter` 接口：`username` → `openid`；`register_date_range` → `created_at_start` / `created_at_end`
  - `UserStatus` 类型：删除，改用 `boolean`

- [ ] **`src/types/order.ts`**
  - `Order` 接口：`order_no` → `out_trade_no`；`user_name` → `candidate_name`；`user_avatar` → 删除（后端无此字段）；`amount` → `price`；`user_phone` → `candidate_phone`
  - `OrderDetail` 接口：同上 + `refund_reason` / `refund_amount` 等确认后端有无
  - `OrderFilter` 接口：`user_phone` → 改用后端已有的筛选参数

- [ ] **`src/types/content.ts`**
  - `ContentItem` 接口：`zone` → `zone_type`；`sort_weight` → `sort_order`；`external_url` → `link_url`；`status` → `is_active: boolean`
  - `Banner` 接口：`status` → `is_active: boolean`
  - `Course` 接口：`name` → `title`；`status` → `is_active: boolean`；`class_count` → 从 `batches` 计算或删除；`schedules` → 暂标记可选（列表不返回）
  - `ContentStatus` 类型：删除，改用 `boolean`

- [ ] **`src/types/quiz.ts`**
  - `Question` 接口：`content` → `question_text`；`type` → `question_type`；`correct_answer: string[]` → `correct_answer: string`；`options: [{label,content}]` → `options: Record<string, string>`
  - `QuestionFilter` 接口：`keyword` → 对齐后端 query 参数名

- [ ] **`src/types/dashboard.ts`**
  - 逐字段对照后端 `AdminStatisticsService.dashboard()` 实际返回，调整字段名

### 常量

- [ ] **`src/core/constants.ts`**
  - `USER_STATUS_MAP`：`{ active: ..., banned: ... }` → `{ true: { color: 'green', text: '正常' }, false: { color: 'red', text: '已禁用' } }`
  - `CONTENT_STATUS_MAP`：同上改为布尔键
  - `STATUS_OPTIONS`：`[{ label: '上架', value: 'online' }, ...]` → `[{ label: '上架', value: true }, { label: '下架', value: false }]`
