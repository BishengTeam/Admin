# 后台用户管理 — 接口调用流程

> 日期：2026-06-09

---

## 路由总览

prefix: `/admin/users`，权限: `user:list` / `user:write` / `user:delete`

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/admin/users` | 用户列表（分页+筛选） | `user:list` |
| GET | `/admin/users/export` | 导出 CSV | `user:list` |
| GET | `/admin/users/{user_id}` | 用户详情 | `user:list` |
| PUT | `/admin/users/{user_id}` | 编辑用户 | `user:list` |
| PATCH | `/admin/users/{user_id}/status` | 启用/禁用 | `user:list` |
| POST | `/admin/users/batch-delete` | 批量软删除 | `user:delete` |
| GET | `/admin/users/{user_id}/orders` | 用户订单列表 | `user:list` |
| GET | `/admin/users/{user_id}/conversations` | AI 对话记录 | `user:list` |
| PUT | `/admin/users/{user_id}/identity/review` | 审核实名认证 | `user:write` |

---

## 接口调用流程

```
进入用户管理页
  │
  ├── GET /admin/users
  │     ?openid=&phone=&created_at_start=&created_at_end=&page=1
  │     分页用户列表，支持按 openid / 手机号 / 注册时间筛选
  │
  ├── GET /admin/users/export
  │     筛选条件同上，返回 CSV 文件下载
  │
  ▼
点击某用户 → 用户详情
  │
  ├── GET /admin/users/{user_id}                       ← 基本信息
  │     响应: id, openid, phone, is_active, created_at
  │
  ├── GET /admin/users/{user_id}/orders                ← 订单记录
  │     响应: list[AdminUserOrderBrief]
  │
  ├── GET /admin/users/{user_id}/conversations          ← AI 对话记录
  │     响应: list[AdminUserConversationBrief]
  │
  ▼
操作区
  │
  ├── PUT  /admin/users/{user_id}                      ← 编辑用户
  │     请求体: { "is_active": true }
  │
  ├── PATCH /admin/users/{user_id}/status               ← 启用/禁用开关
  │     请求体: { "is_active": false }
  │
  ├── PUT  /admin/users/{user_id}/identity/review       ← 🆕 审核实名认证
  │     请求体: { "status": "verified", "comment": "人工审核通过" }
  │     status 取值: verified / rejected
  │     verified 时自动写入 verified_at
  │
  └── POST /admin/users/batch-delete                   ← 批量软删除
        请求体: { "ids": [1, 2, 3] }
        限流: 3 次/分钟
        实现: user.is_deleted = True
```
